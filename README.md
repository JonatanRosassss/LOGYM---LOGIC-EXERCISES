#LOGYM

Una aplicacion que proporciona ejercicios de logica.

En el principio para no perder de vista y desorientarse se empezo usando nombres de variables largas y bien detalladas. 
Con el paso esto cada vez era mas insostenible y daba verguenza preentar una variable que ocupe toda una linea, asi que se refactorizo los nombres de variable y funciones.
